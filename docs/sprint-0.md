# Sprint 0: Project Foundation

Sprint 0 establishes a maintainable local development foundation:

- React + TypeScript + Vite frontend.
- FastAPI backend.
- SQLite through SQLAlchemy 2.x.
- Alembic migration setup.
- Health-check API and frontend connection state.
- Professional dark workbench shell.
- Basic backend and frontend tests.

Out of scope:

- AI Agent workflows.
- Image, keyframe, or video generation.
- ComfyUI integration.
- Upload flows.
- Login, cloud sync, or multi-user features.
- Infinite canvas or 3D director stage.
