# Local Drama Studio Web

React + TypeScript + Vite frontend for the Local Drama Studio workbench.

## Run

```powershell
npm install
npm run dev
```

Open `http://localhost:5173`.

## Test

```powershell
npm run typecheck
npm run test
```

## Routes

- `/projects`: project list.
- `/projects/:projectId`: project detail placeholder.

User-visible copy is centralized under `src/locales` with `zh-CN` as the active default.
